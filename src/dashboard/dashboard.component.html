Shopping Application
Welcome to the Shopping application.

This is a sample application to showcase end-to-end data flow between the Angular 2 UI, .NET WebApi2, nHibernate into a MySql DB and back. The DevExtreme grid is incorporated and its use (along with the whole application's) is entitlement based off the Entitlements Service. SignalR is used to show realtime edits roundtripping.

Please use the navigation bar to begin.
